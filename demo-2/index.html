Course 1: MEAN Stack

MEAN is a free and open-source JavaScript software stack for building dynamic web sites and web applications. The MEAN stack is MongoDB, Express.js, AngularJS (or Angular), and Node.js. Because all components of the MEAN stack support programs that are written in JavaScript, MEAN applications can be written in one language for both server-side and client-side execution environments.

Course 2: MERN Stack

MERN is a free and open-source JavaScript software stack for building dynamic web sites and web applications. The MERN stack is MongoDB, Express.js, ReactJS, and Node.js. Because all components of the MERN stack support programs that are written in JavaScript, MERN applications can be written in one language for both server-side and client-side execution environments.
